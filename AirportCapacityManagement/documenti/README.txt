Capacity Management per gli aeroporti dello stato di New York


Requirements:
-Java version 1.8 o successive.
-MySQL versione 5.5


Test:
 1- Fare il download del progetto da Github all'indirizzo https://github.com/TdP-prove-finali/VaraldoArianna/tree/master
  o fare il checkout direttamente.
 2-importare il progetto da Eclipse seguendo le istruzioni
 3-nella cartella db scaricare lo zip del dataset e importarlo nel server mySQL
 4-avviare l'applicazione da Eclipse facendo il Run as Application sulla classe Main.java 
 nel package it.polito.tdp.ariannavaraldo.aircapmgt
 5-utilizzare l'applicazione
 
 (video tutorial su YouTube  https://youtu.be/YSG2NKjYj8A)
